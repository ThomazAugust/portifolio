Algumas palavras sobre mim

Olá, sou o Thomaz Augusto! Cientista da computação por formação, atraido por tecnologia, curioso por natureza, em constante busca por novos aprendizados e, melhores soluções para resolução dos problemas que me são apresentados.

Formação

Sou graduado em Ciência da Computação pela Universidade Tiradentes - UNIT desde 2017.

Conhecimentos

Durante minha experiencia de trabalho tenho atuado em projetos utilizando conceitos como SOLID e Clean Code, com tecnologias C#, .Net Core, .Net framework, Kafka, Rabbit, TDD, DDD, Entity Framework, Dapper, XUnit, Opencover, DevOps, Jenkins, git(GitHub, GitHub Actions, GitLab) bitbucket, arquitetura orientada a eventos, padrões arquiteturais de proejtos(Hexagonal, microsserviços, CQRS, Ports And Adapters, Onion, Cascata) no SQL DB (mongo, redis), SQL Server e desenvolvimento ágil com Scrum, Kanban e suas cerimônias.

Objetivos
